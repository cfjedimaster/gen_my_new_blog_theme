Mastering CSS Grid for Modern Web Layouts

Published on October 27, 2023
Categories:
  • CSS
  • Frontend
  • Tutorial

CSS Grid is a two-dimensional layout system for the web. It lets you lay content out in rows and columns, and has many features that make building complex layouts straightforward. It's a powerful tool that, once mastered, will fundamentally change how you approach CSS layouts.

Unlike Flexbox, which is largely a one-dimensional system, Grid is two-dimensional. This means it can handle both columns and rows at the same time, which is perfect for website layouts, as opposed to just component layouts.

Getting Started with `display: grid`

To start using Grid, you need a container element. You apply `display: grid` to this container, and then you can define your columns and rows.

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* Three equal-width columns */
  grid-template-rows: auto; /* Row height based on content */
  gap: 20px; /* Space between grid items */
}

In the example above, `grid-template-columns` defines the track sizing of the grid columns. The `fr` unit is a fractional unit, and `1fr` for each column means they will each take up an equal share of the available space.

Positioning Items

You can explicitly place items on the grid using `grid-column-start`, `grid-column-end`, `grid-row-start`, and `grid-row-end`, or the shorthand properties `grid-column` and `grid-row`.

.grid-item-1 {
  grid-column: 1 / 3; /* Span from column line 1 to 3 */
  grid-row: 1;        /* Place in the first row */
}

This level of control is what makes Grid so powerful for creating layouts that were previously difficult to achieve with CSS, often requiring complex workarounds or JavaScript.

We've only scratched the surface here. Dive into properties like `grid-template-areas`, `minmax()`, and `auto-fit` to unlock even more potential. Happy coding!
